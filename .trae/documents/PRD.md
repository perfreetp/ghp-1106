## 1. 产品概述

王者荣耀回合制对战版是一款基于经典MOBA英雄IP改编的桌面回合制策略游戏，面向喜欢策略排兵的玩家群体。

- 项目将MOBA英雄以回合制战棋形式呈现，玩家通过阵容搭配、站位布局、技能时机决策进行对战
- 核心玩法价值：降低MOBA操作门槛，强化策略深度，满足喜欢回合制策略爱好者的细分需求

## 2. 核心特性

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 本地存档自动创建账号 | 全部游戏功能 |

### 2.2 功能模块

1. **主菜单**：开始游戏、英雄图鉴、阵容编辑、关卡地图、商店、回放、设置、退出
2. **英雄图鉴**：英雄列表、属性面板、技能说明、装备展示
3. **阵容编辑**：英雄选择、拖拽编队、站位调整、铭文装配
4. **战斗棋盘**：回合制对战、行动顺序、技能释放、战斗日志
5. **关卡地图**：剧情关卡、挑战模式、进度保存
6. **商店系统**：英雄购买、装备购买、金币消费
7. **回放系统**：战斗录像、逐帧回放、倍速播放
8. **设置面板**：音效音量、画质调节、分辨率、存档管理

## 3. 核心流程

### 3.1 游戏主流程
玩家从主菜单开始，选择进入阵容编辑搭配英雄与站位，在关卡地图选择关卡，进入战斗棋盘进行回合制对战，战斗结束获得奖励返回主菜单循环。

```mermaid
flowchart TD
    A["主菜单"] --> B["阵容编辑"]
    B --> C["关卡地图"]
    C --> D["战斗棋盘"]
    D --> E{"胜负判定"]
    E -->|胜利 --> F["奖励结算"]
    F --> G["商店（可购买/升级"]
    E -->|失败 --> H["返回阵容调整"]
    H --> B
    F --> A
    A --> I["英雄图鉴"]
    A --> J["回放系统"]
    A --> K["设置"]
```

### 3.2 战斗回合流程
```mermaid
flowchart TD
    A["回合开始"] --> B["行动顺序排序（按速度）"]
    B --> C["当前英雄行动"]
    C --> D{"操作选择"}
    D -->|普通攻击 --> E["选择目标"]
    D -->|技能释放 --> F["检查能量/冷却"]
    D -->|防御 --> G["防御姿态"]
    D -->|待机 --> H["跳过回合"]
    E --> I["计算伤害/效果"]
    F --> I
    G --> I
    H --> I
    I --> J{"胜负判定？"}
    J -->|否 --> K["下一位英雄"]
    K --> B
    J -->|是 --> L["战斗结束"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色玄幻风格，深蓝+金色为主色调，营造高贵史诗感
- **次色调**：血条红色、技能蓝色、能量黄色
- **按钮风格**：3D浮雕质感、金色边框、悬浮发光效果、点击凹陷反馈
- **字体**：标题使用书法风字体、正文使用黑体
- **布局风格**：卡片式布局、面板分区、玻璃拟态效果
- **图标风格**：游戏手绘风图标、技能图标统一风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主菜单 | 导航区域 | 居中布局、悬浮动画、粒子背景、3D按钮 |
| 英雄图鉴 | 列表+详情 | 左右分栏、卡片悬停放大、属性进度条 |
| 阵容编辑 | 拖拽区+棋盘 | 拖拽效果、网格高亮、连线预览 |
| 战斗棋盘 | 战场+操作 | 六边形/方格战场、行动条动画、技能特效 |
| 关卡地图 | 地图节点 | 世界地图背景、节点连线、解锁进度 |
| 商店 | 商品卡片 | 商品网格、购买弹窗、价格标签 |
| 回放 | 列表+播放器 | 时间轴、倍速控制、步骤列表 |
| 设置 | 选项面板 | 滑块、开关、存档列表 |

### 4.3 响应式
- 桌面端优先：1920x1080最佳体验
- 支持自适应：最小1280x720可玩
- 非移动端适配

### 4.4 动画与特效
- 战斗技能释放：粒子爆炸、屏幕震动、伤害飘字
- 英雄移动：网格平滑过渡动画
- 菜单切换：淡入淡出+滑动过渡
- 按钮交互：悬浮放大、光效流动
